## ğŸ› Troubleshooting

**Blank page after deployment?**
- Verify `base` in `vite.config.js` matches your repo name (with leading and trailing slashes)
- Check browser console for errors
- Make sure GitHub Pages is enabled in Settings

**404 on routes?**
- GitHub Pages doesn't support client-side routing natively
- The app uses hash-based routing (`/#/dashboard`) for compatibility

**Build failing?**
- Check the Actions tab for error logs
- Ensure `frontend/package-lock.json` exists
- Try building locally first with `npm run build`
